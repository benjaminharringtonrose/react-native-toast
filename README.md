# react-native-toast

native toast library

## Installation

```sh
npm install react-native-toast
```

## Usage

```js
import ToastModule from "react-native-toast";

// ...

<Button
  title="Show Toast"
  onPress={() => ToastModule.show('Yip yip')}
/>
```
